Animated GIF solutions:

- [LICEcap](http://www.cockos.com/licecap/ "simple animated screen capture tool for Windows and OS X")
- [Recordit](http://recordit.co/ "OS X and Windows")
- [gifify](https://github.com/jclem/gifify "A nice little shell script that converts any .MOV file to a GIF")
- [GIF Scrubber](https://chrome.google.com/webstore/detail/gif-scrubber/gbdacbnhlfdlllckelpdkgeklfjfgcmp "A Chrome extension for controlling GIFs like you would a movie")
- [OS X Screencast to animated GIF (gist)](https://gist.github.com/dergachev/4627207 "This gist shows how to create a GIF screencast using only free OS X tools: QuickTime, ffmpeg, and gifsicle.")
